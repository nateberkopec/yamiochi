# Yamiochi

Yamiochi is a minimal, RFC-compliant HTTP/1.1 Rack server for Ruby, created autonomously by a [dark factory](https://simonwillison.net/2026/Feb/7/software-factory/).

Yamiochi is specified by humans, but is 100% implemented by LLM agents.

It is designed to run behind a reverse proxy and uses a prefork worker model.
